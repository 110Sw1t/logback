Download links

Logback modules are available as downloads including full source code, class files and documentation.

    logback-${pom.version}.zip

    logback-${pom.version}.tar.gz

If you wish to download an older version of logback, please visit the distributions directory.

A console plugin for Eclipse is also available. It allows you to recieve logging events in a convenient Eclipse view, and offers, among other functionnalities, the powerful filtering framework present in logback. A more precise description can be found in the plugin's user guide.

  • logback Console Plugin for Eclipse
